Options

In every last parameter you can set the following options. 

{
	size: {
		width: 300,
		height: 100
	},
	offset: {
		x: 0,
		y: -100
	},
	title: null,
	scroll: true,
	useEscKey: true,
	disposeOnClose: true,
	closeButton: true,
	focus: true,
	fx: {
		type: 'tween',
		open: 1,
		close: 0,
		options: {
			property: 'opacity',
			duration: 400
		}
	},
	okText: 'Ok', // Only MooDialog.Alert, MooDialog.Confirm,MooDialog.Error and MooDialog.Prompt
	cancelText: 'Cancel' // Only MooDialog.Confirm /*,
	onOpen: $empty,
	onClose: $empty,
	onShow: $empty,
	onHide: $empty*/
}
